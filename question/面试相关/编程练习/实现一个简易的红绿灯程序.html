    # 实现一个简易的红绿灯程序

    • 描述：

      使用js编写一个程序，实现红、黄、蓝三种状态循环打印。

    • 示例：

    • 实现：

      1. 利用setInterval()定时显示状态
       let n = 0
        function officeLight(){
            let status = ''
            if(n % 3 === 1) {
                status = 'red'
            } else if (n % 3 === 2) {
                status = 'green'
            } else {
                status = 'yellow'
            }
            n++
            console.log(status)
        }
        setInterval(() => {
            officeLight()
        }, 3000)
