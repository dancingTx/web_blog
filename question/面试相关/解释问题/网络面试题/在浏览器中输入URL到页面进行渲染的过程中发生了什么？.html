    # 在浏览器中输入URL到页面进行渲染的过程中发生了什么？

    可以参考如下图理解：（图片来自《HTTP权威指南》）

    描述：

    • 当用户输入URL后，浏览器开始解析主机名
    • DNS进行域名解析，即 将语义化的主机名解析成IP地址
    • 浏览器通过解析的域名获得端口号
    • 浏览器根据得到的IP地址和端口号发起TCP连接
    • 浏览器发起HTTP请求
    • 浏览器读取服务器返回的响应报文
    • 浏览器对返回的HTML进行渲染
    • 浏览器断开TCP连接
